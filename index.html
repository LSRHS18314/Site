About The Class
This page was made in a Web Publishing class at Lincoln-Sudbury Regional High School. In this class, we do a variety of assignments that give us a much better understanding of how web pages are created and what needs to be done to make a physically appealing site. I came into this class with absoultely no knowledge on the topic, and now am able to create and publish a webpage on my own.
Interactive Story

Found in the dropdown menu, the interactive story project has you make a series of selections that lead to a certain result of the story. Each selection sparks a new question, which leads to a different result.

Images

Found in the navbar menu, the image project explores how images are cited and prepared for use on the web.

Citizenship

Found in the navbar menu, the citizenship project explores the key qualities in a good citizen and some examples of perfroming these actions.

Store

Found in the dropdown menu, the store wesbite project uses the features of bootstrap to simulate a real life online store.
