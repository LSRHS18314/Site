About The Class
This page was made in a Web Publishing class at Lincoln-Sudbury Regional High School. In this class, we do a variety of assignments that give us a much better understanding of how web pages are created and what needs to be done to make a physically appealing site. I came into this class with absoultely no knowledge on the topic, and now am able to create and publish a webpage on my own.
